# Collaboration & Contribution Guidelines

Welcome to the AI Driven Development Course community! To maintain a productive and organized environment, please follow these guidelines.

## 👥 Roles & Responsibilities

### 👨‍🏫 Instructors
- **Guidelines**: Provide course materials in the `Instructor/` folder.
- **Reviews**: Review Pull Requests in the `Students/` and `Collaborative-Projects/` folders.
- **Issue Management**: Monitor and label issues (e.g., `student-help`, `instructor-review`).

### 🎓 Students
- **Personal Folder**: Each student MUST create a folder in `Students/` named `Name-SapId` (e.g., `JohnDoe-12345`).
- **Project Structure**: Within your folder, create a new sub-folder for each project.
- **Commits**: Use descriptive commit messages (e.g., `feat: Add basic web scraper to Project 1`).
- **Issues**: Raise issues for questions or to propose new project ideas.

---

## 🏗️ Collaborative Projects
Collaborative projects are located in `Collaborative-Projects/`.
- **Engagement**: Anyone can contribute by opening an issue or a Pull Request.
- **Coordination**: Before starting significant work, please comment on an existing issue or open a new one with the label `collaboration-wanted`.
- **Quality**: Ensure all code is documented and follows basic linting standards.

---

## 🛠️ Raising Issues & Pull Requests

### Raising Issues
- Use the templates provided when creating a new issue.
- Assign appropriate labels (see `README.md` for the list).
- Provide a clear title and detailed description.

### Pull Requests
- All changes must be submitted via Pull Request.
- Pull Requests to your own student folder should still be reviewed if possible, especially for final project submissions (tag the instructor).
- Pull Requests to `Collaborative-Projects/` require at least one approval from another student or the instructor.

---

## 🏷️ Issue Tags

| Tag | Purpose |
| :--- | :--- |
| `bug` | Technical problems in collaborative projects. |
| `student-help` | Questions about the course or specific tasks. |
| `project-proposal` | Suggesting a new project idea. |
| `collaboration-wanted` | Seeking partners for a group project. |
| `instructor-review` | Requesting final review for a project submission. |
